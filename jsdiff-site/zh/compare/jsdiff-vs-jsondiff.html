JSDiff vs JSON Diff：选哪个 JSON 对比工具？

JSDiff 与 JSON Diff（jsondiff.com）都提供 JSON 对比。JSDiff 额外提供多种 diff 模式（行、词、字符、补丁）、完全本地处理以保护隐私，以及离线支持。JSON Diff 侧重简洁界面与基础对比。若需要隐私、离线或多于纯 JSON 的 diff 能力，JSDiff 更合适；若只需偶尔做最简对比，JSON Diff 也可考虑。

何时选哪个

选 JSDiff：需要本地处理与隐私、离线使用、多种 diff 模式或移动端友好对比时。

选 JSON Diff：只需极简界面做偶尔的 JSON 对比，且不需要离线或额外模式时。
