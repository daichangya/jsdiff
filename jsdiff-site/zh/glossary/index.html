词条表：JSON Diff 与相关术语

与 JSON 对比、差异算法及 JSDiff 等工具相关的核心术语与概念。每条词条包含清晰定义及其与 JSON/代码对比的关系。

词条

JSON Diff

比较两个 JSON 文档，找出新增、删除或修改的键与值。与逐行文本 diff 不同，JSON diff 可以是语义的——忽略属性顺序、关注结构。

Myers 算法

一种 O(ND) 的高效算法，用于计算两个序列之间的最短编辑脚本。被 Git、JSDiff 及众多文本编辑器用于 diff 与合并。

查看定义 →

语义 Diff

按含义而非原始文本进行比较。对 JSON 而言，若两个对象具有相同的键与值则视为相等，与属性顺序无关。
